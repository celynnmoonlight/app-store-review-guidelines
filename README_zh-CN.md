# 应用商店审核指南

本仓库包含主流应用商店的审核指南，已转换为 Markdown 格式，方便阅读和跟踪变更。

## 概述

本项目提供了三大应用商店（Apple App Store、Google Play、华为 AppGallery）的审核指南和开发者政策，采用便捷的 Markdown 格式呈现。对于希望确保应用符合各商店上架要求的移动应用开发者来说，这些文档是必读内容。

## 现有指南

| 应用商店 | 文档 | 原始 PDF |
|---------|------|----------|
| Apple App Store | [Apple App Store 审核指南](apple_app_store_review_guidelines.md) | [PDF](apple_app_store_review_guidelines.pdf) |
| Google Play | [Google Play 开发者计划政策](google_play_developer_program_policies.md) | [PDF](google_play_developer_program_policies.pdf) |
| 华为 AppGallery | [华为 AppGallery 审核指南](huawei_appgallery_review_guidelines.md) | [PDF](huawei_appgallery_review_guidelines.pdf) |

## 使用方法

1. **阅读指南**：浏览 Markdown 文件，了解各商店的审核要求
2. **跟踪变更**：使用 Git 跟踪指南随时间的变化
3. **对比政策**：查阅并比较不同应用商店的政策差异
