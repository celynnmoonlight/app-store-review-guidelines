# App Store Review Guidelines

This repository contains review guidelines for major app stores, converted to Markdown for easier reading and tracking changes.

## Overview

This project provides the review guidelines and developer policies from the three major app stores in a convenient Markdown format. These documents are essential reading for mobile app developers who want to ensure their apps meet the requirements for listing on Apple App Store, Google Play, and Huawei AppGallery.

## Available Guidelines

| Store | Document | Original PDF |
|-------|----------|--------------|
| Apple App Store | [Apple App Store Review Guidelines](apple_app_store_review_guidelines.md) | [PDF](apple_app_store_review_guidelines.pdf) |
| Google Play | [Google Play Developer Program Policies](google_play_developer_program_policies.md) | [PDF](google_play_developer_program_policies.pdf) |
| Huawei AppGallery | [Huawei AppGallery Review Guidelines](huawei_appgallery_review_guidelines.md) | [PDF](huawei_appgallery_review_guidelines.pdf) |

## Usage

1. **Read the guidelines**: Browse the Markdown files to understand the review requirements for each store
2. **Track changes**: Use Git to track changes in guidelines over time
3. **Compare policies**: Review and compare policies across different app stores

## Important Notes

- These Markdown documents are converted from the original PDF sources
- The PDF files contain the authoritative and most up-to-date versions
- Please refer to the official developer portals for the latest guidelines:
  - [Apple App Store Review Guidelines](https://developer.apple.com/cn/app-store/review/guidelines/)
  - [Google Play Developer Program Policies](https://support.google.com/googleplay/android-developer/answer/16852659)
  - [Huawei AppGallery Review Guidelines](https://developer.huawei.com/consumer/cn/doc/app/50104)
